# POL2610 Materials

This repository stores the working materials for `POL2610`.

## Layout

- Root directory: original PDF readings.
- `mineru/`: structured MinerU parse outputs for each PDF.
  - `full.md`: Markdown extraction.
  - `layout.json`: page/layout metadata.
  - `content_list.json`: structured content blocks.

## Notes

- MinerU temporary ZIP archives and duplicate exported source PDFs are intentionally omitted.
- The parsed result directory names match the source PDF filenames for direct lookup.
