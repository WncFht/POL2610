# Korea / Topic 1 Resource Archive

整理日期：2026-04-19

这个 `resource/` 目录围绕以下主线整理：

> 美国开发的 LLM 在韩国最重要的影响，不只是提高效率，而是把 AI 从生产率工具变成了主权问题。

主线逻辑：
1. 韩国面临低增长、老龄化、劳动力压力，因此快速吸收生成式 AI。
2. 美国 LLM 的扩散不只是“被使用”，还把韩国更深地嵌入美国主导的 AI 模型—算力—芯片—数据中心生态。
3. 这种嵌入带来依赖与风险，因此韩国一边继续合作，一边通过 `AI Basic Act` 与 `Sovereign AI Foundation Model` 项目追求更强的主权与治理能力。
